**Heart Disease Prediction using SVC and PCA**
_This project predicts the presence of heart disease using a Support Vector Classifier (SVC). 
Principal Component Analysis (PCA) is applied for dimensionality reduction and visualization of the data_.

**Features**
* Data preprocessing and train-test split
* Support Vector Classifier (SVC) for classification
* Principal Component Analysis (PCA) for dimensionality reduction
* PCA-based data visualization
* SVC + PCA classification
* Model evaluation using a confusion matrix
  
**Technologies Used**
~ Python
~ Google Colab
~ NumPy
~ Pandas
~ Matplotlib
~ Scikit-learn
~ Results
The model's performance is evaluated using a confusion matrix to analyze classification accuracy.
